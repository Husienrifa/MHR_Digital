# Stock Screening Chatbot Project

## Overview
The Stock Screening Chatbot is a project designed to help users filter and identify stocks based on specific criteria. It provides an interactive interface where users can ask questions about stocks and receive instant feedback based on market data.

## Features
- **Natural Language Processing**: Understand user queries in plain English.
- **Stock Filter**: Apply various filters such as market cap, P/E ratio, and dividend yield.
- **Real-Time Data**: Access current stock data to provide up-to-date information.
- **User Interface**: A simple and intuitive interface for seamless interaction.

## Technologies Used
- **Programming Language**: Python
- **Libraries**: nltk, pandas, numpy, flask
- **APIs**: Alpha Vantage or Yahoo Finance API for market data.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Husienrifa/MHR_Digital.git
   ```
2. Change directory to the project folder:
   ```bash
   cd MHR_Digital
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```

## Usage
1. Start the application and open the UI in your web browser.
2. Enter your stock criteria in the input field.
3. Receive results based on your query.
